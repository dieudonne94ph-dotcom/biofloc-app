Présentation du système

Ce système est un outil de pilotage complet de votre exploitation aquacole Biofloc. Il centralise en un seul endroit la gestion biologique (qualité d'eau, croissance des poissons), la gestion logistique (stocks de probiotiques) et la gestion financière (comptabilité en Franc CFA).

Chaque bassin est suivi individuellement. Les mesures de terrain (Température, pH, Oxygène dissous, Ammoniaque) sont optionnelles — saisissez uniquement ce que vous mesurez ce jour-là. La cible de croissance est de +2.9 g par jour. Le fût de probiotiques est de 100 litres.

Procédures journalières
1
Chaque matin — avant le tour des bassins
Rendez-vous dans l'onglet 🖨️ Fiche Collecte et imprimez le tableau. Cette fiche vous accompagne sur le terrain pour noter à la main les mesures de chaque bassin.
2
Après le tour — saisie dans Config Bassins
Reportez pour chaque bassin : l'aliment réellement servi (kg), et si mesurés : température, pH, DO, NH₃. Cliquez sur 💾 Enregistrer par ligne.
3
Vérifier la logistique probiotiques
Consultez l'onglet 🧴 Logistique. Si la jauge devient rouge (stock < 15 L), préparez immédiatement un nouveau batch de 100 L et mettez à jour le stock.
4
Tous les 15 jours — pesée de contrôle
Prélevez un échantillon représentatif par bassin. Mettez à jour le Poids Moyen dans Config Bassins. Ajoutez la mesure dans 📈 Suivi Croissance pour suivre le GMQ et le FCR.
5
Lors d'une récolte
Passez le statut du bassin à "Récolté" dans Config Bassins et enregistrez. Ses données apparaîtront automatiquement dans l'onglet 🏆 Récoltes.
Niveaux d'accès
Responsable
Accès complet à tous les onglets incluant Comptabilité et Récoltes. Peut supprimer tout bassin, même avec données. Reçoit les notifications de modifications opérateurs.
Opérateur
Accès à tous les onglets sauf Comptabilité et Récoltes. Peut supprimer un bassin uniquement sans données enregistrées. Toute modification sur des données antérieures génère une notification au Responsable.
Légende des couleurs
Vert — Cellule de saisie autorisée. Modifiable par l'opérateur ou le responsable.
Gris discret — Champ optionnel (Temp, pH, DO, NH₃). Saisissez seulement si mesuré.
Orange — Alerte modérée nécessitant votre attention : température basse, pH hors plage, stock en baisse.
Rouge — Alerte critique, action immédiate requise : NH₃ > 0.5, DO < 4, FCR > 1.5, stock < 15 L.
Or — CUMP (Coût Unitaire Moyen Pondéré) calculé automatiquement depuis vos lots d'achat.
Les 5 types d'aliment

Chaque bassin se voit attribuer un type d'aliment selon le stade de croissance. Saisissez vos lots d'achat dans l'onglet 🌾 Aliments — le CUMP est calculé automatiquement.

🥚
Granulé 1 < 5 g Larve / Post-larve
Très petits granulés pour les premières semaines d'élevage. Phase critique de démarrage.
💡 Soyez attentif à la qualité de l'eau pendant cette phase.
🐟
Granulé 2 5 – 20 g Juvénile
Début de croissance active. Le poisson commence à prendre du poids rapidement.
💡 Vérifiez le taux d'alimentation régulièrement.
🐠
Granulé 3 20 – 150 g Grossissement
Principal aliment de production. Stade le plus long — optimisez le FCR pour réduire les coûts.
💡 Type le plus utilisé — stockez en quantité suffisante.
🐡
Granulé 4 200 – 400 g Pré-finition
Poissons en forte prise de masse. Aliment plus dense en énergie pour la phase finale.
💡 Surveillez la biomasse et ajustez les rations.
🎣
Granulé 6 400 – 600 g Finition
Phase finale avant récolte. L'objectif est d'atteindre le poids de commercialisation optimal.
💡 Préparez la récolte dès que le poids cible est atteint.

CLÉS DE CONFIGURATION ESP32
Project ID: biofloc-system-fc6f5
Farm Code: —
Collection: farms/{farmCode}/sensors/{bassinName}
Structure Firebase prête — format des données attendues

Voici le format JSON exact que votre ESP32 devra envoyer à Firebase. Cette structure est déjà créée et attend les données.

// Collection : farms/{farmCode}/sensors/{sensorId}
{
  "bassin": "Bassin 01",
  "sensorId": "esp32_bassin01",
  "timestamp": "2025-01-15T08:30:00Z",
  "temp": 26.4, // °C — DS18B20
  "ph": 7.82, // pH — Sonde Atlas EZO
  "DO": 6.51, // mg/L — Atlas Scientific DO
  "NH3": 0.12, // mg/L — Sonde ammoniac
  "batterie": 87, // % batterie capteur
  "frequence_min": 15 // intervalle lecture (minutes)
}
Quand un capteur envoie des données, la plateforme mettra automatiquement à jour les valeurs du bassin correspondant en Config Bassins et déclenchera les alertes si nécessaire.
